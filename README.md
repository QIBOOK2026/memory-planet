# 3D 照片记忆星球

这是一个本地浏览器作品生成器：上传照片后，可以生成一颗可旋转、可展开、可查看照片故事的 3D 记忆星球。当前版本以“星球”为核心，可以创建多颗独立的记忆星球，并导出为可分享的电脑 3D HTML 作品包。

## 使用流程

1. 在顶部 Tab 中点击“星球”，创建或进入一颗记忆星球。
2. 点击“上传”，填写当前相册信息并上传照片；后续上传会追加到当前相册后面，不会覆盖其他相册。
3. 点击“主题”，选择科技蓝、暖金纪念、粉紫心动、星空白蓝。
4. 点击“形态”，选择星球、银河、爱心、记忆环、时间波纹。
5. 点击“故事”，在照片列表中编辑照片名称和故事。
6. 点击“预览”，展开照片故事，悬停照片会锁定并放大，点击可打开详情。
7. 点击“导出”，生成电脑 3D HTML、手机动态 Canvas 版，或备份/导入 JSON 数据。

首版照片上限为 200 张。照片越多，导出的 HTML 文件会越大，打开速度也会变慢。

## 星球结构

当前界面只保留星球概念：

- 星球首页：进入“星球”Tab 后，会看到多颗记忆星球漂浮在 3D 空间中，可拖动旋转、滚轮缩放。
- 记忆星球：每颗星球拥有独立标题、照片、故事、主题、形态和音乐设置。选中星球后可点击底部“打开”或面板里的“进入星球”加载详情。

所有星球共用同一套星球引擎。以后新增自动播放、更多手势、新形态等功能时，只需要升级星球引擎，不需要给每颗星球单独复制代码。

## 打开方式

推荐在本目录启动本地网页服务后访问：

```powershell
python -m http.server 5177
```

然后打开：

```text
http://localhost:5177/
```

导出的电脑 3D HTML 会保留当前作品数据，适合在电脑浏览器中展示。手机动态版会生成轻量 Canvas 星球，更适合发到手机上拖动查看。导出的作品包优先支持直接双击打开；如果浏览器限制了某些能力，可以同样用本地网页服务打开导出的 HTML。

## 交互

- 拖动画面：360 度旋转记忆星球
- 鼠标滚轮：拉近或拉远视角
- 单指触控：旋转
- 双指触控：缩放
- 展开后悬停照片：锁定并放大
- 点击照片：查看大图和故事
- 空格键：展开/收回

## 备份与恢复

- 点击“备份”会下载 JSON 文件，里面包含作品标题、主题、形态、照片、故事、声音和高级视觉设置。
- 点击“导入备份”可以恢复之前导出的 JSON，适合换设备、换浏览器或误删后恢复。
- 点击“清空”前会二次确认；如果作品重要，建议先备份。

## 高级设置

高级设置中保留粒子密度、爆发强度、荧光半径、旋转速度、帧率上限、渲染清晰度、氛围音乐、交互音效和实验手势控制。粒子密度、爆发强度、荧光半径、旋转速度、帧率上限和渲染清晰度会随本地自动保存、JSON 备份、导入和电脑 3D HTML 导出一起恢复。手势功能需要联网加载 MediaPipe Hands，并允许摄像头权限；开启后可随时点击“关闭实验手势”释放摄像头。

## 声音

- 点击“开启氛围音乐”后，会根据当前主题生成一层 Web Audio 合成氛围声。
- “内置曲库”提供几首已下载到 `assets/music/` 的 Wikimedia Commons 开放授权曲目，也保留本地合成氛围作为兜底。
- 展开、收回、切换形态、打开照片和收藏照片都有轻量交互音效。
- 可以用“声音音量”调节整体音量，也可以单独关闭交互音效。

本地内置曲目包含 Placid Ambient by MusicLFiles、Scott Buckley - Sleep、Scott Buckley - The Long Dark、Spiritual Ambient by MusicLFiles。使用这些曲目分享作品时，请保留页面中的作者/授权提示。

可用实验手势：

- 一指：伸出食指移动手掌，拨动星球旋转。
- 张开手掌：展开照片故事。
- 握拳：收回星球。
- 捏合：拇指和食指靠近，展开时打开当前/最近照片。
- OK：拇指和食指靠近，其他手指伸出，收藏或取消收藏当前照片。
- V 手势：伸出食指和中指，切换到下一种星球形态。
- 左右挥手：张开手掌左右移动，切换上一张/下一张故事。
- 双手拉开/合拢：放大或缩小记忆星球。
